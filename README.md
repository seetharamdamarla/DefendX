# DefendX - Automated Vulnerability Detection Engine

DefendX is a high-performance, full-stack security assessment platform designed to automate the detection of modern web vulnerabilities. It provides a unified command center for developers and security professionals to monitor, analyze, and remediate security risks in real-time.

## üöÄ Key Features

*   **Automated Scanning**: Comprehensive detection of OWASP Top 10 vulnerabilities, including SQL Injection, XSS, and misconfigurations.
*   **Security Operations Center (SOC)**: A premium, unified dashboard for managing security tasks, risks, and targets.
*   **Real-time Threat Monitoring**: Interactive charts and live metrics for keeping track of your application's health.
*   **Premium SaaS Aesthetic**: Built with a sleek, 3D glassmorphic design focused on clarity and professionalism.
*   **Secure Authentication**: Role-based access control with session persistence.

## üì¶ Installation & Setup

### Prerequisites
- Node.js (v16+)
- Python (v3.9+)

### 1. Clone the Repository
```bash
git clone <your-repository-url>
cd DefendX
```

### 2. Backend Setup
```bash
cd backend
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

### 3. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```
